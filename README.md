## Getting Started

### Installing

These instructions will get you up and running with a copy of the project on
your local machine for development and testing purposes.

1. Fork and Clone this repo
2. Install all the required packages in the Gemfile with the command: `poetry install`

## Running the tests 

**Testing with Pytest**
To run the full test suite you will need to run the command:
`pytest`
